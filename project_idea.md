
### Project Idea
```
Project — Personal Study Assistant Agent
What it does: An AI agent that helps users study a topic by generating quizzes, summarizing documents (PDFs/URLs), and tracking progress.

Stack: 
- Python — core language; use httpx, pypdf2, or requests for content fetching 
- Microsoft Agent Framework — multi-turn conversation loop 
- MCP Server — custom Python MCP server exposing tools: fetch_url_content, generate_quiz, save_progress 

Touches agent orchestration, tool creation, file I/O, and prompt design in one cohesive project.

Estimated effort: ~10 days
```
